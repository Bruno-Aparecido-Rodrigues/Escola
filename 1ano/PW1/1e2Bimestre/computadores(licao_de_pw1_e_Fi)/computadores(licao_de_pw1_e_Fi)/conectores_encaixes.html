1. Identifique e explique todos os conectores externos frontal e traseiro do computador.

Entrada e saída de áudio: Como o nome diz serve para a entrada e saída de áudio.
portas USB: É utilizado para a conexão de vários dispositivos sendo 1.0 com 1,5 Mb/s 2.0 com 480Mb/s 3.0 com 4.8Gb/s.
VGA: Serve para saída de vídeo, conexão para tv ou monitores.
PS2: Utilizada para a conexão de teclados e mouses antigos.
P2: Utilizado para a conexão de fones.
P3: Para microfone com ou sem fone.
MIDI port: facilita a comunicação em tempo real entre instrumentos musicais eletrônicos, computadores e dispositivos relacionados.
LAN: conexão do cabo de rede.
Paralelo port: utilizada antigamente para impressoras.
Serial comunications: Servia para a utilização de outros dispositivos (antecessor do USB), no futuro foi substituído pelo mesmo.
Line in: serve para conectar um dispositivo de áudio externo.


12. Quais os tipos de conectores para unidade de armazenamento interna?
Os conectores sao o Pata e Sata que possuem a mesma função(transferência de dados entre dispositivos de armazenamento em massa).

13. O que é slot de expansão? Dê seus tipos e comente cada um brevemente.

Os slots de expansão servem para que seja possível adicionar recursos à sua placa-mãe. Neles você conecta placas de rede, placa de som, modems, placa de captura.
PCI: Serve para conectar Dispositivos como placa de vídeo.
AGP: Antigo conector para placas de vídeo.
AMR: Para conectar Modem on Board.
CNR: Para o Modem on Board também.
ISA: Antigo conector de dispositivos.
EISA: Era utilizado para Placas de vídeos potentes.
VESA: Sucessor do EISA.
PCIe: Serve para diversos periféricos para computador.
